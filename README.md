# TaskFlow

A minimal React to-do list app. Built by Aditya Sahni for INSPIRE 1PL3 SMART Goal Project.

## Features
- Add tasks
- - Delete tasks
  - - Mark tasks complete
    - - Stats dashboard
     
      - ## Tech Stack
      - - React 19 + Vite
        - - Pure CSS
         
          - ## Run Locally
          - ```
            npm install
            npm run dev
            ```
